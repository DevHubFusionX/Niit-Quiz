# Study Guide PDFs

Place your PDF study guides in this directory with the following naming convention:

- `PLT_StudyGuide.pdf` - Programming Logic & Techniques
- `WebDev_StudyGuide.pdf` - Introduction to Web Development  
- `IWCD_StudyGuide.pdf` - Internet & Web Client Development
- `Dreamweaver_StudyGuide.pdf` - Adobe Dreamweaver
- `MySQL_StudyGuide.pdf` - MySQL Database
- `PHP_StudyGuide.pdf` - PHP Programming

The app will automatically show download buttons for subjects that have PDF files available.
